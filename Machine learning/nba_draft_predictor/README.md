Predicts if a college basketball player will become a successful NBA player (hit) or a bust based on college stats.

## Features Used
- Points Per Game (PPG)
- Rebounds Per Game (RPG)
- Assists Per Game (APG)
- Height (inches)
- Age at draft
- Shooting percentages

## How to Run
1. Install dependencies: `pip install -r requirements.txt`
2. Run: `python main.py`
3. See predictions, graphs, and accuracy!

Made with Python + scikit-learn
